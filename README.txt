Source for Java package 'com.deetysoft.config.'

'Config' is a package of utility classes for viewing configuration data as
a collection of properties backed by property files.

Config project structure is the Maven default and a pom file is included.

Config requires Java JDK 1.8 or higher.

Install Maven, cd to the project root and run 'mvn verify' to build and run the unit tests.

Config unit tests are TestNG tests.

Build the Javadocs to see package and class documentation.

Run 'mvn package' to build a jar file to include in your CLASSPATH.
